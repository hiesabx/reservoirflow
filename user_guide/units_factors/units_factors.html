Units & Factors üìê#

ReservoirFlow has a fixed unit system used when an object is initiated (i.e., object for the modules such as: grids, fluids, models). The value of unit property can be set to "field", "metric", or "lab". By default, unit="field" is used. This system was inspired by Abou-Kassem et al.[1]. This system was inspired by Aziz and Settarib1 and Abou-Kassem et al.b2.

Warning

As of now, changing class unit will not automatically convert values. In the near future, a unit converter will be developed to match units and easily convert units from one system to another.
