Collaboration 🤝
================

Here we list all of our active collaborations with universities, companies, teams, and individuals.

How to collaborate?
-------------------

You need to register for a collaboration, so we can assess our resources before we can give you our commitment. To request a collaboration, please check `Register 🔑 </research_development/collaboration/register.html>`_.
